About:
PTNJ Booking System is web application where user can book a snorkeling place and find every snorkeling places that are available in Malaysia. This website was done using the Laravel framework.
Main Page:
The main page of the applicaion is where users who can login are able to search a snorkeling location and proceed to choose a set of date with the number of people in order to book. The website also supports multiple language which users can change on the footer based on the picture below:
Location Details Page:
Users can know more about the locations that they are interesting in going and straight up proceed with booking by a click of a button.
Administrator Page:
The admin page where admins are able to see, edit and manage the user list, agency list, location list, booking from user list and generate report based on a set of dates.
