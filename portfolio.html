Project Name:
Kiasu Perks
Project Description:
A mobile application that let users to get discount coupons to be used in many different shops & services all around the world.
About:
Kiasu Perks as a whole consists of four parts which is the mobile app, the admin management page, the merchant page and the landing page. The mobile app is for any user to use while the admin portal allows merchants to post and manage the discount coupons into the mobile app.
Mobile App:
The mobile app is where users can enjoy discounts from all around the world by claiming the promotional codes that are available inside of the app.
On the first version of the app, the team and I developed the app in Unity but was eventually scrapped as the app was too laggy and suffers several performance issues. To solve this problem, the chief suggested for us to recreate the app using Ionic Framework which have been maintained until the official release of the app.
Admin Management Page:
The admin management page will allow the admin to add and manage the merchants. Moreover, admin can also control which discount counpons be shown in the mobile app.
Merchant Management Page:
The merchant management website will give access to merchants for them to create and manage their own promotions that will appear on the mobile app.
The websites are developed using Bootstrap as the UI and basic PHP without any library. Moreover, AJAX is used in calling the API to show and manipulate the data on the webpage.
Landing Page:
This is the promotional website for users to know more information about the mobile app.
